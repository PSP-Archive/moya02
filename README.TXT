が、多分何とかなるんじゃないのかなぁ。sqrtuの辺りなんだけど。
ちとやってみたが、解からなくなって元に戻した。（光源計算部）

（そんな変なことは真似したら駄目だよ）不審に思ったらみてね。

それから、なごなご跳ねるのが止まらないのは精度不足のせい。
